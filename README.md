# marvellous terraform vpc plugin

## Spin up basic network setup on AWS Cloud

- VPC
- Subnets (public, private)
- Internet Gateway
- Routing table

Everything with Stackspot!

## Pre requisites

- AWS Cli authentication profile works properly
- Terraform

*tested in Ubuntu 20.04*
